# Rails Simple CRUD

# Requirements

* Ruby 2.5.3
* Ruby 5.2.1
* PostgreSQL 9.5.14

# How to use

* rails new <nameapps>      | create new rails project
* cd <nameapps>             | move to directory project
* rake db:create            | create database
* rails server              | start a web server
* running at http://localhost:3000
